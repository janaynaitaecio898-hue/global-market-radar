全球资产雷达

Agent 接入

像 AIHOT 一样，把信息流变成 Agent、RSS 和 API 都能读取的市场决策数据源。

市场雷达技能

给 Agent 使用的技能入口：自动读取精选事件、按资产和周期过滤，并输出短中长期影响判断。

使用全球资产雷达：
1. 获取今日精选影响因素
2. 按股票、债券、黄金、商品、汇率分类
3. 输出当前市场主线和风险触发条件

RSS 订阅

给人和自动化工具使用的轻量订阅源。可以分为精选、全部动态、日报、黄金、债券、风险事件。

/rss/featured.xml /rss/daily.xml /rss/gold.xml

REST API

GitHub Pages 正式版 v1 先提供只读 JSON 数据文件。后续迁移到后端时，再升级为带查询参数、限流和 token 的 REST API。

GET /data/signals.json GET /data/assets.json GET /data/scenarios.json GET /data/market_snapshot.json

X 财经账号池

正式版 v1.7 支持通过 X API 抓取四类账号池：官方媒体、宏观投资人、市场情绪、中国与亚洲观察。GitHub 仓库添加 X_BEARER_TOKEN secret 后，工作流会自动把最近推文转成市场线索。

secrets.X_BEARER_TOKEN from:Reuters OR from:business from:Sino_Market OR from:NikkeiAsia
正式版 v1 已启用静态 JSON 数据层；后续可以继续升级 RSS、REST API 和 Agent Skill。
